Crope - Creative Web Agency HTML Template

  • latest update: 21/06/2020
  • by: george_fx

Crope is a powerful, clean, creative and easy to use Web Agency HTML Template. It is highly suitable template for companies that offer UI/UX Design, Branding, SEO&SMM, Digital Marketing, IT services, Development Services, Portfolio and all other internet marketing and services related companies.

Overview

  • css - contains css files which were generated from scss files
  • images - contains site images
  • js - contains site javascripts
  • webfonts - contains fonts used on the site

Structure

Sublime Text 3 (download for free). Used to edit the .HTML, .JS, and .CSS files.

The following scheme displays the template files structure.

  • root - contains the .html files of the template.

- 
    webfonts-  contains  all fonts.
    css -  contains all the .css files used in the design.
    images -  contains all the images used in .html files.
    js -  contains JavaScript files.
